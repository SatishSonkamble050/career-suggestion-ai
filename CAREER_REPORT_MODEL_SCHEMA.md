# Career Report Model & Schema Documentation

## Overview

The Career Report system stores comprehensive career guidance reports generated by the LangGraph workflow for each student, including all analyses and recommendations.

---

## Database Table Structure

### Table: `career_reports`

```sql
CREATE TABLE career_reports (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    student_id INTEGER NOT NULL UNIQUE,
    
    -- Analysis sections (JSON)
    academic_analysis JSON,
    skill_analysis JSON,
    interest_analysis JSON,
    preference_analysis JSON,
    market_analysis JSON,
    
    -- Final report
    final_report TEXT NOT NULL,
    
    -- Report metadata
    title VARCHAR(255),
    summary VARCHAR(500),
    
    -- Recommendations (JSON)
    recommended_careers JSON,
    action_plan JSON,
    next_steps JSON,
    
    -- Rating and feedback
    rating FLOAT,
    feedback TEXT,
    
    -- System metadata
    version VARCHAR(10) DEFAULT '1.0',
    model_used VARCHAR(100) DEFAULT 'gpt-3.5-turbo',
    
    -- Timestamps
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    expires_at DATETIME,
    
    -- Foreign key
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE
);
```

## SQLAlchemy Model

### File: `app/models/career_report.py`

```python
class CareerReport(Base):
    """Career guidance report model"""
    __tablename__ = "career_reports"
    
    id: int (Primary Key, Index)
    student_id: int (Foreign Key → students.id, Index)
    
    # Analysis sections (stored as JSON)
    academic_analysis: Dict (Optional)
    skill_analysis: Dict (Optional)
    interest_analysis: Dict (Optional)
    preference_analysis: Dict (Optional)
    market_analysis: Dict (Optional)
    
    # Final report
    final_report: str (Required, Text)
    
    # Report metadata
    title: str (Optional, Max 255)
    summary: str (Optional, Max 500)
    
    # Recommendations
    recommended_careers: List[str] (Optional, JSON)
    action_plan: Dict (Optional, JSON)
    next_steps: List[str] (Optional, JSON)
    
    # Ratings
    rating: float (Optional, 0-5)
    feedback: str (Optional, Text)
    
    # System info
    version: str (Default: "1.0")
    model_used: str (Default: "gpt-3.5-turbo")
    
    # Timestamps
    created_at: datetime (Auto-set to now)
    updated_at: datetime (Auto-set to now, updates on change)
    expires_at: datetime (Optional expiry)
```
